# SEO-GEO Report Conventions

## Language

All responses, analyses and reports are produced **in English by default**. If the user requests French (or any other language), switch to that language for the entire report.

## Scoring format

### Overall score: /90
| Dimension | Max | Level |
|---|---|---|
| Technical SEO (on-page + crawlability) | /10 | 🔴/🟡/🟢 |
| Core Web Vitals | /10 | 🔴/🟡/🟢 |
| Local SEO OR Content & Authority | /20 | 🔴/🟡/🟢 |
| E-E-A-T | /10 | 🔴/🟡/🟢 |
| Entity SEO | /10 | 🔴/🟡/🟢 |
| GEO | /20 | 🔴/🟡/🟢 |
| UX & Conversion | /10 | 🔴/🟡/🟢 |

### Color legend
- 🔴 < 50% of max score
- 🟡 50–75% of max score
- 🟢 > 75% of max score

## Audit report structure
1. Executive summary (score, ranking vs competitors, top 3 strengths, top 3 weaknesses)
2. Score dashboard
3. Detailed results per dimension
4. Competitive analysis
5. Prioritized action plan (Quick Wins / Medium term / Long term)
6. Tracking metrics
7. Glossary
8. Conclusion & next steps

## Implementation report structure
1. Action summary
2. Generated files (with deployment instructions)
3. Implementation checklist
4. Post-deployment verification

## Monitoring report structure
1. Current score vs baseline
2. Positive changes
3. Regressions detected
4. Alerts
5. Recommended actions

## Data reliability
- Always cite sources (URL + date)
- If data is inaccessible → 🔍 "Estimate based on [method]"
- Without GSC/GA4 access → clearly state the uncertainty
- Never fabricate metrics or rankings
